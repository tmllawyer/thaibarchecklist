# ⚖️ Niti Thamma (Thai Bar Checklist)

**Niti Thamma — Thai Bar Checklist** คือเว็บแอปพลิเคชันที่ออกแบบมาเพื่อนักศึกษากฎหมาย ช่วยในการติดตามความคืบหน้าการอ่านหนังสือ ฟังคำบรรยาย และเตรียมตัวสอบเนติบัณฑิต (ภาค 1 สมัยที่ 79) อย่างเป็นระบบ จบครบในที่เดียว

🌐 **Live Demo:** [https://thaibarchecklist.netlify.app](https://thaibarchecklist.netlify.app)

---

## 🌟 ฟีเจอร์เด่น (Key Features)

- **📊 Dashboard & Progress Tracking:** ติดตามความคืบหน้าการอ่านแบบแยกสถิติ ขาอาญา และ ขาแพ่ง พร้อมระบบนับถอยหลังวันสอบ (Countdown)
- **✅ Smart Checklist:** เช็คลิสต์รายวิชา บันทึกสถานะการอ่าน/ฟังคำบรรยาย พร้อมกำหนดระดับความพร้อม (Priority) และบุ๊กมาร์กวิชาที่ต้องทบทวน (Star)
- **📝 Smart Notebook (สมุดจดอัจฉริยะ):** พื้นที่จดสรุปย่อ แยกสัดส่วนเนื้อหา มาตราสำคัญ และเลขฎีกา พร้อมระบบ Export ออกเป็น Markdown หรือ สั่งพิมพ์เป็น PDF
- **📂 Document Manager:** ระบบจัดการลิงก์เอกสารประกอบการเรียน เชื่อมโยงได้ทั้งไฟล์ในเครื่อง (Local), Google Drive และ E-Book ของเนติฯ
- **🎯 Mock Exam & Focus Mode:** โหมดจำลองห้องสอบ พร้อมระบบจับเวลา (Timer), ระบบ Pomodoro, กระดาษเขียนคำตอบ, และ **Zen Mode** สำหรับตัดสิ่งรบกวน พร้อมระบบเทียบ "ธงคำตอบ"
- **☁️ Cloud Sync:** เข้าสู่ระบบอย่างปลอดภัยด้วย Google Account เพื่อซิงค์ข้อมูลความคืบหน้าข้ามอุปกรณ์ ข้อมูลไม่หายเมื่อเปลี่ยนเครื่อง
- **🎨 Customizable Themes:** รองรับโหมดสว่าง (Light Mode), โหมดมืด (Dark Mode) และธีมพิเศษ "ชุดครุย (Barrister Gown Theme)"
- **📱 PWA Ready:** สามารถติดตั้งลงบนหน้าจอโฮม (Add to Home Screen) เพื่อใช้งานผ่านมือถือหรือแท็บเล็ตได้เหมือนแอปพลิเคชันทั่วไป

---

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Styling:** Tailwind CSS (CDN)
- **Backend & Database:** Google Firebase (Authentication & Cloud Firestore)
- **Icons & UI Effects:** Lucide Icons, Canvas Confetti
- **Hosting:** Netlify

---

## 🚀 วิธีการใช้งานเบื้องต้น (Getting Started)

1. เข้าสู่เว็บไซต์ผ่านลิงก์ของโปรเจกต์
2. กดปุ่ม **"ลงชื่อเข้าใช้ด้วย Google"** เพื่อสร้างฐานข้อมูลส่วนตัว
3. เริ่มต้นอัปเดตสถานะรายวิชา, ดึงลิงก์ PDF มาแปะ, หรือจดสรุปคำบรรยายได้ทันที
4. สามารถเพิ่มรายวิชาที่เรียนเสริมเองได้ผ่านปุ่ม "+ เพิ่มวิชา"
